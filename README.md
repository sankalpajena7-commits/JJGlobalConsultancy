## 🚀 How to Start the Application

### Option 1 — Using `live-server` (Recommended)

Make sure you have [Node.js](https://nodejs.org/) installed, then run:

```bash
# Navigate to the project directory
cd JJ_Global_Consultancy

# Start the live dev server (auto-reloads on file changes)
npx live-server --port=3030
```

The site will open automatically at **http://localhost:3030**

---

### Option 2 — Using Python (No install required)

```bash
# Python 3
cd JJ_Global_Consultancy
python3 -m http.server 3030
```

Then open **http://localhost:3030** in your browser.

---

### Option 3 — Open directly in browser

Simply open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).

> ⚠️ Some features (fonts, images via relative paths) may not load correctly when opened as `file://` — use a local server for best results.
